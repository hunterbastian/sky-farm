# Sky Farm Roadmap

## Current State
Floating island with farming, chickens, trees, pond, day/night cycle, CRT retro aesthetic. Economy exists (coins + wood) but nothing to spend it on yet.

---

## Phase 1 — Economy Loop
_Give coins and wood a purpose._

- [ ] **Shop panel** — buy seeds (cost per type), buy chickens, buy decorations
- [ ] **Seed costs** — wheat free, star berry 5, pumpkin 10, moon flower 18
- [ ] **Sell basket** — click harvested crops to auto-sell, show floating +coin numbers
- [ ] **Wood crafting** — fences, paths, bridges, signs from wood
- [ ] **Egg value scaling** — golden eggs (rare, worth 10 coins)

## Phase 2 — Fishing
_Make the pond interactive._

- [ ] **Fishing rod tool** (5th tool slot)
- [ ] **Fishing minigame** — click pond, timing-based catch
- [ ] **Fish types** — cloud trout (common, 5 coins), sky bass (uncommon, 12), star koi (rare, 25)
- [ ] **Fish in pond** — visible swimming pixel fish

## Phase 3 — Building
_Let players shape their farm._

- [ ] **Place fences** from wood inventory (click to place)
- [ ] **Dirt paths** — till grass into paths for aesthetics
- [ ] **Decorations** — scarecrow, lamp post, bench, well
- [ ] **Small house** — cosmetic, maybe enter for shop/storage

## Phase 4 — More Animals
_Expand the farm life._

- [ ] **Buy more chickens** from shop (max 8)
- [ ] **Cat or dog** — follows cursor, purely cosmetic companion
- [ ] **Frogs** — hop around pond edges
- [ ] **Fireflies** — appear at dusk, glow particles

## Phase 5 — Progression
_Give players goals._

- [ ] **Tool upgrades** — copper/silver/gold hoe (larger area), better axe (fewer chops)
- [ ] **Farm level** — XP from harvesting, unlocks new seeds and items
- [ ] **Seasonal events** — cherry blossoms in spring, snow in winter
- [ ] **Island expansion** — spend coins to unlock new land patches

## Phase 6 — Polish
_Make it feel complete._

- [ ] **Sound effects** — Web Audio API: tool clicks, coin jingle, water splash, chicken cluck, ambient nature
- [ ] **Music** — looping chiptune farm track (day) + softer night theme
- [ ] **Save slots** — multiple farms
- [ ] **Tutorial** — first-time hints that fade after use
- [ ] **Mobile support** — touch controls, responsive layout
